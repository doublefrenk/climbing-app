## Why

Il README attuale è il template di default di Vite e non descrive il progetto. Chiunque voglia capire, avviare o contribuire al progetto non ha informazioni su architettura, stack, configurazione locale o intenti dell'applicazione.

## What Changes

- Sostituire il README.md esistente con una documentazione completa del progetto
- Aggiungere descrizione dell'applicazione e degli intenti
- Documentare l'architettura e lo stack tecnologico
- Aggiungere istruzioni per avviare il progetto in locale (frontend, backend, Docker)
- Documentare le variabili d'ambiente richieste
- Descrivere la struttura del progetto
